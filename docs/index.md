# 📘 Financial Powerhouse Platform — Documentation Home

Welcome to the official documentation for the **Financial Powerhouse Platform**, a unified, operator‑grade fintech monorepo combining:

- **Flask Backend API** — compliance, fraud detection, contracts, telemetry  
- **React Native / Expo Mobile App** — full banking client  
- **Shared TypeScript TRPC Server** — auth, OAuth, notifications, LLM, routing  
- **Drizzle ORM Schema + Migrations** — mobile‑side relational modeling  
- **Unified Developer Workflows** — consistent DX across backend + mobile  
- **Cockpit‑Grade Operator Visibility** — telemetry, audit logs, health checks  

This site provides complete architecture, onboarding, operational, and API documentation for the entire platform.
